# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built using **HTML5**, **CSS3**, and **JavaScript**. This project showcases my profile, skills, work experience, and projects in a clean and interactive interface.

---

## 📖 Overview

This portfolio website serves as my personal online presence where I introduce myself, highlight my technical skills, and showcase my professional experience. It is designed with a responsive layout and smooth user interactions to provide an engaging browsing experience.

The project demonstrates core front-end development concepts including semantic HTML, modern CSS styling, Flexbox layouts, and JavaScript-based typing animations.

---

## ✨ Features

* 🎨 Clean and modern user interface
* 📱 Responsive layout
* 🧭 Fixed navigation bar
* 💻 Hero section with introduction
* ⌨️ Animated typing effect using Typed.js
* 💼 Work Experience section
* 🎯 Smooth hover effects
* 🔤 Google Fonts integration
* ⚡ Lightweight and fast loading
* 🧩 Easy to customize and extend

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Typed.js

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/portfolio-website.git
```

### Run the Project

1. Open the project folder.
2. Double-click `index.html` or open it in your preferred web browser.

No additional installation or build tools are required.
